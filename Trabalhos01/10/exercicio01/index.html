Criando sua primeira página HTML

Ilustra as técnicas básicas de programação usando o Visual Studio Code para escrita de HTML.

Objetios da aprendizagem

Depois de concluir este módulo, você poderá:

  • Descrver os conceitos básicos de HTML
  • Usar uma ferramenta de codificação
  • Criar uma página HTML básica
  • Criar listas vinculadas em HTML

Pré-requisitos

Conhecimento básico da terminologia computacional.
